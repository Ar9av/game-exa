# Example games

Complete Phaser 3 games generated by **game-exa** — one per validated genre. Each ships with full source, sprite sheets, level data, and `game-state.json` so you can run `npm install && npm run dev` and play immediately.

| Folder | Genre | Win condition | Lose condition |
|---|---|---|---|
| `crystal-village/` | RPG Overworld | Collect 5 crystals | none (exploration) |
| `slime-slayer/` | Top-down adventure | Collect 3 gems | HP reaches 0 |
| `pixel-pete/` | Platformer | Collect 5 coins | HP 0 or fall off screen |
| `star-defender/` | Shoot-em-up | Destroy 10 asteroids | HP reaches 0 |
| `void-breaker/` | Arena shooter | Survive 10 waves | HP reaches 0 |

## Run any example

```bash
cd examples/crystal-village   # or slime-slayer / pixel-pete / star-defender / void-breaker
npm install
npm run dev                   # opens http://127.0.0.1:5173
```

Controls:
- **Movement**: arrow keys or WASD
- **Interact / Action**: Z or Space (crystal-village: talk to NPCs; others: attack/jump/fire)

## Run the gap-checker against an example

From the repo root:

```bash
# Static check (fast, no browser)
node skills/gap-checker/scripts/static_check.mjs examples/crystal-village

# Full dynamic check (Playwright, 30 s)
node skills/gap-checker/scripts/dynamic_check.mjs examples/crystal-village --port 5199 --seconds 30
```

## Visual notes

`slime-slayer` and `star-defender` use **procedural placeholder sprites** (flat-color cells) — that's what game-exa generates without an image-generation API key, or with `--placeholder-sprites`.

`crystal-village` and `pixel-pete` have hand-crafted procedural pixel-art tilesets and entity sprites. `crystal-village` features full humanoid characters (hero, elder, villager, guard) with a portrait dialogue system — pressing Z near an NPC opens a typewriter dialogue box with their sprite portrait on the left.

To regenerate with real **GPT Image 2** (`gpt-image-2`) assets, set `FAL_KEY` (or `OPENAI_API_KEY`) and re-run:

```bash
gamewright generate "..." --cwd examples/my-game
```
